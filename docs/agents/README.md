# AI agent playbooks

Documents in this folder define how **primary agents**, **subagents** (delegation), and **review agents** should behave when working on this codebase or when **rebuilding a similar project** (see [BUILDING_SIMILAR_PROJECT.md](../BUILDING_SIMILAR_PROJECT.md)).

| File | Purpose |
|------|---------|
| [AGENT.md](AGENT.md) | Full architectural alignment, design patterns, coding standards |
| [SUBAGENTS.md](SUBAGENTS.md) | Specialist roles, boundaries, and delegation triggers |
| [REVIEW_AGENTS.md](REVIEW_AGENTS.md) | Pre-merge review personas and checklists |

The repository root **[AGENTS.md](../../AGENTS.md)** is the short entry point for tools that expect a single top-level file.
